Alternative Withdraw
How does this all work?

zkSync Alternative Withdrawal is way to get funds to Layer 1 without interacting directly with the protocol. zkSync supports most of web3-compatible wallets, so we highly recommend you to use the so the official client to withdraw funds if that is possible as it is cheaper and more convenient.

In order for the account to be eligible for an alternative withdrawal all of the following must be true:
  • It must exist (hold any funds) in the zkSync network for at least 24 hours.
  • The account must be locked (no ChangePubKey operation so far).

The amount to compensate is somewhat higher than the base fee due to technical reasons, but the overhead is negligible (less than 0.000001 ETH).
